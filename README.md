# react-native-webview-ios

React Native WebView for iOS

## Installation

```sh
npm install react-native-webview-ios
```

## Usage

```js
import { WebviewIosView } from 'react-native-webview-ios';

// ...

<WebviewIosView url="https://open.spotify.com" />;
```
